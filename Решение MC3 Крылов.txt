Решение ДЗ 1
Обмануть mc3.exe меняя значение регистров
1. Ставим брейк поинт по адресу 40101D, так как 401000 - это вход другой функции (проверки а не main) - узнали в IDA
2. при отработке к адресу 401042 вводим любое значение (в качестве флага)
3. по адресу 401059 происходит нулевая проверка (длины), вставляем в регистр eax значение "11"  - проходим проверку
4. по адресу 401089 происходит первая проверка (atoi), вставляем в регистр eax значение "14D" (333 в десятичной) - проходим проверку
5. по адресам 4010С2, 4010F8, 40112E происходят три одинковые проверки, вставляем в регистр eax значение "6F" (символ "o" ) - проходим проверки
6. по адресам 401185, 4011DB, 401231, 401287, 4012DD, 401333, 401389, 4013DF, 401435 проходят проверки равенства символов, сравниваются регистры eax и ecx, поэтому устанавливаем в них одинаковые значения, например "1" - проходим эти проверки
7. по адресу 40146A устанавливаем в регистр eax значение "67" (символ "g") - проходим проверку
8. по адресу 4014A0 устанавливаем в регистр eax значение "30" (символ "0") - проходим проверку
Все ! наш флаг верен.
